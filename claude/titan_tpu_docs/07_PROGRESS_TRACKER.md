   - 在 "每日记录" 添加当天条目
   - 更新总体进度条中的状态

2. **完成任务时**
   - 将 ⬜ 改为 ✅

   - 更新里程碑检查

   - 记录到 "每日记录"
   - 同时更新 `08_DEBUG_LOG.md`

4. **开新对话时**
   - 复制最新进度条到对话模板
   - 确保状态同步
